Pickpocket has been updated.

Here's what's new in version 1.2.3.

(This message will only appear when Pickpocket has a significant update.)

New feature

  • You can now open your Pocket links in Pocket's reading-optimized view by pressing Alt or Option while opening a link in the popup. There's also a new setting to make this view the default. If that's enabled, then pressing Alt or Option while opening a link will open it in the normal view.

Keyboard changes on Windows

  • The Alt key modifier is now used to override the default link opening view, as described above.
  • The keyboard shortcut to open all unread links in the popup is now Ctrl+Enter.
  • The keyboard shortcut to archive a link in the popup is now Ctrl+Backspace.
  • The keyboard shortcut to delete a link in the popup is now Shift+Ctrl+Backspace.
  • No keyboard changes on OS X.
